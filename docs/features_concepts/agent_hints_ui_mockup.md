## Keyboard Shortcuts

- **Enter**: Send message (or hint if agent is working)
- **Shift+Enter**: New line in textarea
- **Esc**: Clear input (keeps working state)

**Note**: No keyboard shortcut for Stop - must click button (prevents accidents)
